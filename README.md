# 選択した用語を翻訳する chrome 拡張機能

## 機能

- chrome 上で選択した単語/文章について翻訳する
- 翻訳する言語は以下の４カ国から選択できる
  - 日本語
  - 英語
  - 韓国語
  - 中国語

## 起動方法

- 本リポジトリをクローンする
- `yarn`
- `yarn dev`
- chrome://extensions/ から「パッケージ化されていない拡張機能を読み込む」をクリックし`dist`フォルダを選択
- 任意のページにて単語/文章を選択し、右下のアイコンをクリックすると翻訳される
